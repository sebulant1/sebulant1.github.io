Instructions

  1. Upload an image and then choose a canvas size
  2. Press download and save your .paint file to "%appdata%/.minecraft/paintings"
    • If a "paintings" folder doesn't exist in ".minecraft" then create one yourself
    • If you have changed the install location of your Minecraft then the folder you need to save this file in will change
    • To easily find this folder, if you are unsure, open resource packs in-game and press the "Open Resource Packs Folder" button. Go back one folder and you should be able to make/open your "paintings" folder
  3. Go in-game and hold a full palette in your off-hand and a new canvas of the right size in your main-hand
    • This must be a newly crafted canvas, you cannot take an existing canvas and paint it white
  4. Type the command "/paintimport name" where name is the name of the file you saved (minus the .paint part)
